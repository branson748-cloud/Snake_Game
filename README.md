# Snake_Game
A simple JavaScript, HTML and CSS integrated program that runs a Snake game. The player controls the snake, letting it eat apples and increase his score, trying not to hit the walls.

## Game Controls
The code automatically takes in keyboard inputs using the UP, DOWN, LEFT and RIGHT keys.

## How to play
- Press the START button to begin
- Use the arrow keys to control the snake's movement
- Eat apples (the red circle) to increase your score and let the snake grow
